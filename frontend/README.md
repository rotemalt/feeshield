# FeeShield — Frontend Application 🛡️

**Live Link:** [https://rotemalt.github.io/feeshield/](https://rotemalt.github.io/feeshield/)

This is the React + TypeScript + Vite frontend source code for FeeShield.

### Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build production artifact:
   ```bash
   npm run build
   ```

Please see the [Main Project README](../README.md) at the root level for the complete deployment guide and project details.
