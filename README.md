# FeeShield 🛡️

FeeShield is a premium, open-source B2B SaaS utility designed to audit monthly merchant processing statements and automatically expose hidden processor markups, tiered downgrades, and compliance junk fees. 

**Live Link:** [https://rotemalt.github.io/feeshield/](https://rotemalt.github.io/feeshield/)

It is engineered with a **100% client-side, zero-database architecture** to guarantee absolute privacy: sensitive processing data (like bank routing codes, account digits, business addresses, or names) is parsed and redacted locally in the browser memory before being sent to the AI underwriting agent.

---

## Key Features

*   **🔒 Secure Local PII Redaction:** Runs a browser-side sanitation filter to strip street addresses, phone numbers, contact names, routing numbers, and bank account numbers prior to any API requests.
*   **📂 In-Browser PDF Parsing:** Utilizes `PDF.js` client-side script sandbox to read native credit card statement characters directly in-memory (no server uploads).
*   **🤖 AI Underwriting Agent:** Connects directly to Google Gemini 2.5 Flash Lite using the user's custom API key (stored locally in `localStorage`, never sent to any backend).
*   **📊 Command Center Dashboard:** Vertice.one-inspired enterprise layout featuring:
    *   **Overcharge Threat Level Dial:** Radial risk gauge mapping identified markups.
    *   **Live Redactor Log Streams:** Terminal logs showing active client-side sanitizing steps.
    *   **Contingency Split Pricing Simulator:** DIY vs. DFY savings simulator ($20/mo minimum, capped at $1,000/yr).
    *   **Tailored B2B Negotiation Scripts:** Dynamically writes custom email negotiation copy addressing the processor's retention group.
*   **🚀 Zero-Database Mode:** Disables profile and audit storage. Wiping or refreshing the browser tab completely deletes all trace data from memory.

*   **Core:** React 19, TypeScript 6, Vite 8
*   **Icons:** Lucide React

## Getting Started

### Prerequisites

*   [Node.js](https://nodejs.org/) (v18+ recommended)
*   A Google Gemini API Key (get one for free at [Google AI Studio](https://aistudio.google.com/))

### Installation & Local Dev

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/feeshield.git
    cd feeshield/frontend
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Configure your environment (Optional - you can also input the key directly in the UI):
    Create a `.env` file in the `frontend` folder:
    ```env
    VITE_GEMINI_API_KEY=your_gemini_api_key_here
    ```

4.  Start the local dev server:
    ```bash
    npm run dev
    ```
    Open your browser to the URL displayed in the terminal (typically `http://localhost:5173`).

## GitHub Pages Deployment (100% Free Hosting)

Since FeeShield has no backend database, you can host it entirely for free on GitHub Pages:

1.  Install the GitHub Pages deploy utility:
    ```bash
    npm install gh-pages --save-dev
    ```

2.  Update `vite.config.ts` to include the base path of your GitHub repository:
    ```typescript
    import { defineConfig } from 'vite'
    import react from '@vitejs/plugin-react'

    export default defineConfig({
      plugins: [react()],
      base: '/feeshield/', // replace 'feeshield' with your repository name
    })
    ```

3.  Add deploy scripts to `package.json`:
    ```json
    "scripts": {
      "predeploy": "npm run build",
      "deploy": "gh-pages -d dist"
    }
    ```

4.  Deploy to GitHub:
    ```bash
    npm run deploy
    ```
